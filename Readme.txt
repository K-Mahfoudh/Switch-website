Il faut impérativement lire ce document afin d'assurer une bonne communication entre les developpeurs de ce projet,
les règles suivantes doivent être respectées:

1-Chaque participant à ce porjet doit déclarer la page sur laquelle il va travailler en créant un "New issue".
2-Utilisez des id et des classes qui commence par le nom de la page web que vous allez créer + un nom significatif,
----------------Exemple: class="error404-header" ou bien "forum-footer" / id="forrum-signout-button" ....ect----------------------
3-organisez bien votre code en utilisant l'indentation afin de rendre votre code plus lisible.
4-N'utilisez pas des balises de <style> à l'interieur du code html, il faut créer des fichier css séparer 
5-le nom du fichier css doit être le meme que le fichier html par exemple: forum.html / forum.css
6-Si vous avez des questions ou des problèmes n'hésitez pas à creer un "New issue", expliquez clairement votre problème.

PS: n'oubliez pas d'utiliser Bootstrap

En gros c'est tout, alors Have fun =)
